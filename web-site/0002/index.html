Программа автоматизированного тестирования событий:
Google analytics и Yandex metrika.

Программа позволяет проверить предусмотренные события Google analytics и Yandex metrika отправляемые в момент выполнения каких либо действий на странице сайта.
Сценарии описываются в отдельных файлах в формате json. Для создания и редактирования сценариев предусмотрен удобный редактор.
Программа портативная и кроссплатформенная, с поддержкой операционных систем Windows и Linux.

Системные требования:

  • Windows 7, 8, 10 | Linux (Ubuntu, Mint, и др.)
  • .NET Framework 3.5 SP1
  • Visual C++ Redistributable
  • JRE 8 (Java SE Runtime Environment)

Редактор тестов версия для Windows


Редактор тестов является встроенной вспомогательной программой которая призвана упростить процесс разработки сценариев автоматизированного тестирования.

За выполнение сценариев отвечает программа detect-gaym.jar которая находится в папке bin. Чтобы запустить заранее созданный сценарий автоматизированного тестирования достаточно открыть консоль и ввести команду:

java -jar detect-gaym.jar UTF-8 C:\...\detect-gaym\tests\test-search-ga.json

Под сценарием понимается файл в формате json внутри которого описаны настройки необходимые для работы автотеста, а так же входные данные и шаги их выполнения. Сценарий можно описывать в любом текстовом редакторе но удобнее это делать в специальном редакторе тестов.

Визуальный редактор тестов версия для Windows позволяет не только легко и просто создавать сценарии но так же быстро запускать их на выполнение. Работая в редакторе тестов вам не потребуется запускать сценарий из консоли, достаточно будет нажать одну кнопку и редактор сам сформирует команду для запуска и выполнит сценарий.

Более подробная информация о работе редактора и программы в целом вы найдете в документации.

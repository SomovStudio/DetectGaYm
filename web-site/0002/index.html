Программа автоматизированного тестирования событий:
Google analytics и Yandex metrika.

Программа позволяет проверить предусмотренные события Google analytics и Yandex metrika отправляемые в момент выполнения каких либо действий на странице сайта.
Сценарии описываются в отдельных файлах в формате json. Для создания и редактирования сценариев предусмотрен удобный редактор.
Программа портативная и кроссплатформенная, с поддержкой операционных систем Windows и Linux.

Системные требования:

  • Windows 7, 8, 10 | Linux (Ubuntu, Mint, и др.)
  • .NET Framework 3.5 SP1
  • Visual C++ Redistributable
  • JRE 8 (Java SE Runtime Environment)

Редактор тестов версия для Windows


Редактор тестов является встроенной вспомогательной программой которая призвана упростить процесс разработки сценариев автоматизированного тестирования.

За выполнение сценариев отвечает программа detect-gaym.jar которая находится в папке bin. Чтобы запустить заранее созданный сценарий автоматизированного тестирования достаточно открыть консоль и ввести команду:

java -jar detect-gaym.jar UTF-8 C:\...\detect-gaym\tests\test-search-ga.json

Под сценарием понимается файл в формате json внутри которого описаны настройки необходимые для работы автотеста, а так же входные данные и шаги их выполнения. Сценарий можно описывать в любом текстовом редакторе но удобнее это делать в специальном редакторе тестов.

Визуальный редактор тестов версия для Windows позволяет не только легко и просто создавать сценарии но так же быстро запускать их на выполнение. Работая в редакторе тестов вам не потребуется запускать сценарий из консоли, достаточно будет нажать одну кнопку и редактор сам сформирует команду в файле run-test.bat в папке bin для запуска и выполнит сценарий.

Более подробная информация о работе редактора и программы в целом вы найдете в документации.

Редактор тестов версия для Linux


Визуальный редактор тестов версия для Linux содержит полный набор средств таких же как в версии для Windows для быстрого и удобного описания сценариев в системе Linux.

Все необходимые настройки делаются на форме в специальных полях. Добавление входных данных осуществляетя на вкладке "Данные" в специальной таблице. Для описания шагов действий и команд проверки на вкладке "Шаги" присутствует специальная таблица. Выполнение сценария так же осушествляетя по нажатию на однку кнопку.

Для запуска редактора, файл tests-editor-linux.jar в папке editor обязательно нужно дать права на чтение и запись, а так же включить разрешение на выполнение файла как программы, это делается в свойствах файла на вкладке Права.

За запуск сценария из редактора тестов отвечает файл run-test.sh в папке bin. В этом файле редактор формирует команды необходимые для запуска сценария.

#!/bin/bash
cd /home/.../detect-gaym/bin
java -jar detect-gaym.jar DEFAULT /home/.../detect-gaym/tests/test-search-ga.json

Для запуска файла run-test.sh так же требуется предоставить права на чтение и запись, и включить разрешение на выполнение файла как программу.

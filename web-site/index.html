Бесплатная программа автоматизированного тестирования событий: Google analytics и Yandex metrika.

Программа позволяет проверить предусмотренные события Google analytics и Yandex metrika отправляемые в момент выполнения каких либо действий на странице сайта.
Сценарии описываются в отдельных файлах в формате json. Для создания и редактирования сценариев предусмотрен удобный редактор.
Программа портативная и кроссплатформенная, с поддержкой операционных систем Windows и Linux.

Системные требования:

  • Windows 7, 8, 10 | Linux (Ubuntu, Mint, и др.)
  • .NET Framework 3.5 SP1
  • Visual C++ Redistributable
  • JRE 8 (Java SE Runtime Environment)
  • JDK 8 (Java SE Development Kit)

Релиз назначен на: 12.12.2020

Редактор тестов версия для Windows


Редактор тестов это встроенная вспомогательная программа которая призвана упростить процесс разработки сценариев автоматизированного тестирования.

За выполнение сценариев отвечает программа detect-gaym.jar которая находится в папке bin. Чтобы запустить заранее созданный сценарий автоматизированного тестирования достаточно открыть консоль и ввести команду:

java -jar detect-gaym.jar UTF-8 C:\...\DetectGaYm\tests\test-search-ga.json

Под сценарием понимается файл в формате json внутри которого описаны настройки необходимые для работы автоматизированного теста, а так же входные данные и шаги их выполнения. Сценарий можно описывать в любом текстовом редакторе но удобнее это делать в специальном редакторе тестов.

Визуальный редактор тестов версия для Windows позволяет не только легко и просто создавать сценарий но так же быстро запускать его на выполнение. Работая в редакторе тестов вам не потребуется запускать сценарий из консоли, достаточно будет нажать одну кнопку и редактор сам сформирует команду в файле run-test.bat в папке bin и запустит выполнение сценария автоматизированного тестирования.

Более подробную информацию о работе с программой вы найдете в документации.

Редактор тестов версия для Linux


Визуальный редактор тестов версия для Linux содержит полный набор средств таких же как в версии для Windows для быстрого и удобного описания сценариев в системе Linux.

Для запуска редактора, файл tests-editor-linux.jar в папке editor, обязательно нужно включить права на чтение и запись, а так же включить разрешение на выполнение файла как программы, это делается в свойствах файла на вкладке "Права".

В редакторе тестов все необходимые настройки сценария делаются на вкладке "Общие параметры". Добавление входных данных осуществляетя на вкладке "Данные" в специальной таблице. Для описания шагов действий и команд проверки на вкладке "Шаги" присутствует специальная таблица. Выполнение сценария так же осушествляетя по нажатию на однку кнопку.

При запуске сценария редактор тестов формирует команды и записывает их в файл run-test.sh в папке bin.

#!/bin/bash
cd /home/.../DetectGaYm/bin
java -jar detect-gaym.jar UTF-8 /home/.../DetectGaYm/tests/test-search-ga.json

Для корректной работы файла run-test.sh так же требуется предоставить права на чтение и запись, и включить разрешение на выполнение файла как программу.

Обновление веб драйвера


Программа DetectGaYm способна выполнять действия над элементами страницы в окне браузера используя для этого Selenium инструмент автоматизации действий веб-браузера. Для его работы используется специальный WebDriver под браузер Google Chrome.

Периодически браузер Google Chrome обновляетя и в результате этого WebDriver перестает корректно работать с браузером. В этом случае необходимо зайти на официальную страницу ChromeDriver - WebDriver for Chrome и скачать архив с последней стабильной версией Latest stable release:

  • chromedriver_win32.zip - архив с драйвером для системы Windows
  • chromedriver_linux64.zip - архив с драйвером для системы Linux

Далее необходимо распаковать драйвер из скаченного архива в папки:

  • DetectGaYm\driver\windows - для системы Windows
  • DetectGaYm/driver/linux - для системы Linux

В системе Linux кроме обновления драйвера в программе DetectGaYm так же необходимо обновить драйвер в системе. Для этого нужно выполнить команды:

unzip chromedriver_linux64.zip
sudo mv chromedriver /usr/local/bin/chromedriver
sudo chown root:root /usr/local/bin/chromedriver
sudo chmod +x /usr/local/bin/chromedriver
chromedriver --version

Более подробную информацию о работе с программой вы найдете в документации.

Запуск тестов из командной строки


Программа DetectGaYm позволяет запускать сценарий автоматизированного теста из консоли. В системе Windows это CMD (command line interpreter), а в системе Linux это Terminal (gnome terminal)

В системе Windows чтобы запустить заранее созданный сценарий нужно открыть консоль, перейти в папку bin и запустить выполнение сценария:

cd C:\...\DetectGaYm\bin
java -jar detect-gaym.jar UTF-8 C:\...\DetectGaYm\tests\test-search-ga.json

В системе Linux чтобы запустить заранее созданный сценарий так же нужно открыть консоль, перейти в папку bin и запустить выполнение сценария:

cd /home/.../DetectGaYm/bin
java -jar detect-gaym.jar UTF-8 /home/.../DetectGaYm/tests/test-search-ga.json

Данный способ запуска сценариев удобно использовать в программах предназначенных для обеспечения процесса непрерывной интеграции, таких как Jenkins или TeamCity. Вам нужно будет всего лишь приписать команды запуска.

В визуальном редакторе тестов в меню "Поддержка" присутствует специальная функция которая позволяет формировать команды запуска для редактируемого сценария.
